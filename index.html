AIDLC for VS Code

VS Code Extension

The Builder UI for your AI pipelines. Edit workspace.yaml visually, follow active runs in the sidebar, and open the AIDLC Monitor for token usage and live agent observability.

  • Visual Builder for .aidlc/workspace.yaml
  • Sidebar with live run states & commands
  • AIDLC Monitor — tokens, sessions, agents
  • Session-insights dashboard from Claude transcripts

AIDLC CLI

Terminal · npm

The standalone engine — no VS Code required. Scaffold workspaces, apply pipeline presets, drive runs end-to-end via Claude, and watch everything live from your shell.

  • aidlc init + presets: sdlc, code-review, release-notes
  • aidlc run exec --auto-approve — unattended runs
  • aidlc watch / tail — live run table & stream
  • aidlc dashboard — browser UI on localhost
npm install -g aidlc Source on GitHub

AIDLC Term

macOS App · Rust + wgpu

A GPU-accelerated terminal built for agent work: file explorer that follows your shell, a manager for every Claude Code session on your machine, and a live token-usage dashboard.

  • GPU rendering (wgpu) — tabs, splits, IME-perfect input
  • Agent Manager — status, live tails, focus / interrupt
  • Usage Dashboard — burn rate & limit prediction
  • File tree that follows the shell's cwd, both ways
